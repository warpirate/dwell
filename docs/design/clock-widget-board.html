Build spec — RemoteViews-safe

TypeFraunces (bundled .ttf in res/font — static layout fonts inflate fine; downloadable fonts do not). Time weight 400, optical size large. Date: Inter/sans-serif-medium, uppercase, +0.18em tracking, 11–12sp.

Card depthVertical gradient drawable #2A2620 → #17140F (kills the flat look). 1px top hairline highlight via layer-list. 26dp radius. Drop shadow is the launcher's; we fake lift with the gradient + hairline.

TranslucencyBake ~92% alpha into the gradient colors so the wallpaper warms through. No live blur (RemoteViews can't) — the baked warm tint reads as glass without it.

AccentExactly one green mark per widget — a 6dp dot (ImageView + oval drawable) or a 2dp rule. Never a fill. AA green is #6E9576 on the warm card.

SizingTime numerals tabular + lining so the clock never reflows. Editorial 4×2, the square layouts 2×2, Hairline 4×1. resizeMode horizontal|vertical.

GatingFree = Editorial/cream, hardcoded. Premium flips color/layout/size via the saved WidgetStyle. Config screen shows the controls locked (already built) so the upgrade sells itself.
